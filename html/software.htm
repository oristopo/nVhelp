loading firmware on macOS or linux

Power nanoVNA off, jumper boot pins, power on.

$ cd to folder
$ dfu-util -v -d 0483:df11 --alt 0 -D nanoVNA_0.1.0_ch.dfu
Remove nanoVNA boot jumper and restart it.
